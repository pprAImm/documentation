# documentation

## Кратко

Модуль предоставляет функции для подключения к PostgreSQL, выполнения запросов и управления транзакциями.
Управляет метаданными: пользователи, контент, подписки.  

## Деплой

Требуется драйвер для Go и PostgreSQL

## Основное
Центральное хранилище метаданных и управление состояниями в стриминговой платформе. Он не хранит видео/аудио файлы, а только информацию о пользователях, контенте, подписках, лайках, истории просмотров и т.д.
