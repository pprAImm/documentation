# documentation

## 📌 Суть проекта
Приложение собирает ИИ-видеоролики и сериалы с платформ TikTok и YouTube, чтобы пользователям было удобно смотреть все интересные сгенерированные ролики на одной платформе. 

---

## 🏗 Структура репозиториев организации

Система построена на микросервисной архитектуре. Каждый модуль изолирован в отдельном репозитории:

1. 🔗 [gateway](https://github.com/pprAImm/gateway.git) — единая точка входа, авторизация (JWT) и маршрутизация трафика.
2. 🔗 [core-backend](https://github.com/pprAImm/core-backend.git) — бизнес-логика каталога, метаданные фильмов, оценки, комментарии и работа с БД.
3. 🔗 [streaming-service](https://github.com/pprAImm/streaming-service.git) — высокопроизводительный стриминг тяжелого видеоконтента короткими чанками.
4. 🔗 [frontend](https://github.com/pprAImm/frontend.git) — клиентский интерфейс, плеер вертикальных видео и бесконечная лента.
5. 🔗 [database](https://github.com/pprAImm/database.git) — репозиторий управления базой данных (SQL-скрипты, миграции структуры таблиц, ER-диаграммы).
